# Building the project 
- `mkdir build && cd build && cmake .. -DCMAKE_BUILD_TYPE=Debug && make`
- `make ats` - builds ats
- `make run_tests` - builds tests
- `make benchmark_ats` - builds benchmarking binary
- 
# Running ATS after the build
- `cd build && ./ats`

# Running tests
- `cd build && ./run_tests`

# Running benchmarks
- `cd build && ./benchmark_ats`
